Consolebox

The Consolebox is a lightweight, toolbox application that runs in the terminal! It has a few features that can come in handy for everyday situations.

Current features:

1. Notes
   The notes tool is a quick and easy way to save and access notes you may need. All notes are stored in .txt format, and are opened and edited through your system's default text editor, since editing text in a console would be a nightmare.
2. Conversions
   Conversions offers some quick, simple unit conversions. If you quickly need to figure out how many meters are in a mile, conversions has you covered! It offers conversions for time, distance, computer storage, and weight. All you do is enter the amount of the unit your converting from, and then type the names of the units you are converting from and to!
3. To-Do List
   Similar to notes, but these display right in the console, with deadlines and titles for each To-Do item. Once you finish a task you stored in the To-Do list, you can easily delete it to make space for new tasks.
4. Clock
   The clock just displays your local device time, because why would you not need this.

All of the menus in the console box are navigated used the listed number keys for each menu. This allows for quick and easy access throughout the various menus.
